11 ## Options ##
12 
13 - **search-length** (unsigned integer, default 8): size of each hash table bucket.
14  (A higher setting means more compression quality and slower compression.)
15 - **block-size** (unsigned integer, default 65536): number of hash table buckets.
16  (Make this lower if you're only compressing short texts; a higher setting means
17  higher memory consumption.)
