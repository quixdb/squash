1 # bzip2 Plugin #
2 
3 The bzip2 plugin provides an interface for libbzip2.

12 ## Options ##
13 
14 ### Encoder ###
15 
16 - **level** (integer, 1-9, default 6): Set block size to 100k ... 900k. 1 will
17  result in the fastest compression while 9 will result in the
18  highest compression ratio.
19 - **work-factor** (integer, 0-250, default 30): From the [bzip2
20  documentation](http://www.bzip.org/1.0.5/bzip2-manual-1.0.5.html):
21  > Parameter workFactor controls how the compression phase behaves
22  > when presented with worst case, highly repetitive, input data. If
23  > compression runs into difficulties caused by repetitive data, the
24  > library switches from the standard sorting algorithm to a
25  > fallback algorithm. The fallback is slower than the standard
26  > algorithm by perhaps a factor of three, but always behaves
27  > reasonably, no matter how bad the input.
28 
29 ### Decoder ###
30 
31 - **small** (boolean, default false): Causes the decoder to use a
32  decoding algorithms which requires less memory, but takes about
33  twice as long.
